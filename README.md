## Project Description

**Team Task & File Manager** is a web application that allows teams to create, manage, and track tasks with priorities, deadlines, and statuses, while optionally attaching files such as images or PDFs.  
The application is built with a Flask backend, uses Azure cloud services for data and file storage, and is deployed automatically to Azure App Service using GitHub Actions CI/CD.
